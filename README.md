## Features
- Extracts URLs with parameters from target website
- Saves results to output file automatically
- Color coded terminal output
- Custom User-Agent support

## Installation

git clone https://github.com/ambuj9004-blip/paramspider_lite.py
cd paramspider_lite.py
pip install requests colorama

## Usage

python paramspider_lite.py https://example.com
